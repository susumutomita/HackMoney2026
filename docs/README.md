## プロジェクト概要

ZeroKey Treasuryは、**自律金融のための実行ガバナンスレイヤー** - マルチチェーン、エージェント駆動の世界で暗号通貨取引を保護するAI統治ファイアウォールです。

### 解決する問題

x402時代では、AIエージェントが自律的に支払いを開始できます。これにより以下のリスクが発生します：

- プロンプトインジェクション攻撃
- 設定エラー
- ポリシー違反
- 資金を枯渇させる無限ループ

### ソリューション

```
リクエストソース（AI/人間/x402）
         │
         ▼
   AIポリシーエンジン（Claude API）
         │
         ▼
   オンチェーンガード（ZeroKeyGuard.sol）
         │
         ▼
   ブロックチェーン実行
```

すべてのトランザクションは、実行前にセマンティック分析とオンチェーン強制を通過します。
